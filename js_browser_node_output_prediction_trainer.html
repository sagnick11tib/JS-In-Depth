How to solve

Output Prediction Strategy

Use one repeatable method. First mark synchronous code. Then mark special queues. Then consider host environment differences between browser and Node.js.

01

1. Sync first

Normal function calls, Promise executors, EventEmitter emit, and top-level logs run immediately on the call stack.

2. Microtasks next

Promise callbacks, await continuations and queueMicrotask run after current sync work and before the next task.

3. Node special queue

In Node CommonJS, process.nextTick usually runs before promise microtasks. Do not confuse this with browser behaviour.

4. Tasks / phases

Timers, DOM events, setImmediate, I/O callbacks and rendering callbacks depend on the host event loop and phase.
